# RCS_mini-project
This repo contains the reproduction of the research paper "Robust H∞ Static Output Feedback Control for TCP/AQM
Routers Based on LMI Optimization". 
case3.m file contains the main code which reproduces the results discussed in "section 4" of the "7-Robust...." file.
validation.m file contains the monte carlo simulation analysis of the results.
extension.m file contains the comparison of H∞ SOF AQM controller with PI AQM controller.
